  Breaking the Cipher

    Encryption and decryption are interesting, but breaking the cipher is what matters.  This can be done using exhaustive search (brute force) or statistical analysis methods.  Exhaustive search is useful when the key size is small enough.  Statistical analysis is interesting because it can reveal key length when used correctly; however in this assignment, the key length is known so sophisticated statistical analysis is not required.  If interested in decrypting ciphertext WITHOUT knowing key length, Al Sweigart's book walks through a more comprehensive approach in chapter 18.  Regardless of sophistication, both statistical analysis procedures require frequency analysis of the key characters.  Certain letters occur more frequently in the English language than others.  Below is a frequency table expressed in decimal form.
