   Good evening Tim, this post involves analyzing the adduser shellcode sample from msfvenom.  Msfvenom is a command line tool by Rapid7 that generates payloads in various formats and encoding schemes.  According to Offensive Security, it combined functionally of both msfpayload and msfencode as of June 8, 2015 and placed both tools under a single framework.


Getting Started

   Using any machine with Metasploit Framework, dump available payloads and verify ../adduser exists.  The specific command is msfvenom -l payloads | grep "linux/x86":


Next step is to determine what the payload requires.  Dump the summary information with the command msfvenom -l linux/x86/adduser --list-options.  Note that the basic options take priority.  According to the summary, the adduser payload creates a user, metasploit, with password, metasploit, and adds it to the admin group.
